# SystemPlus

Module to extend functionalities of the System section

## Commands

<ul>
  <li>Export Variables </li>
  <li>Set Multiple Variables</li>
  <li>Run Backup</li>
  <li>Clean Vars</li>
  <li>Random</li>
  <li>Foreground App</li>
  <li>Get Handle from Open windows</li>
</ul>
